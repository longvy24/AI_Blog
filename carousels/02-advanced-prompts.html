The Advanced Ten

Past "Act As An Expert"

01 / 10Context
Context Feeding
"Here's my context: [paste docs]. Answer using ONLY the text above: ..."
Grounds ChatGPT in your facts — it answers from your data instead of guessing.
Full breakdown →
02 / 10System
System Prompts
"From now on: act as my copy editor. Tone: punchy. Never use emojis or fluff."
Set the rules once and they hold all chat — stop repeating your instructions.
Full breakdown →
03 / 10Examples
Few-Shot Prompting
"Sort as + or –. 'Loved it' = +. 'Broke fast' = –. Now sort: ..."
A few labeled examples beat any instruction — the model just copies the pattern.
Full breakdown →
04 / 10Variety
Temperature Control
"Give me 10 wildly different angles. Be bold — I want range, not safe."
Ask for range and you unlock creativity; ask for "the best" and you get vanilla.
Full breakdown →
05 / 10Format
Output Formatting
"Return ONLY a markdown table. Columns: Task | Owner | Due. No intro."
Name the exact shape — table, JSON, bullets — and it comes back clean every time.
Full breakdown →
06 / 10Reasoning
Tree of Thought
"List 3 approaches. Weigh each one. Then expand only the strongest."
Makes ChatGPT explore options before committing — far fewer first-guess misses.
Full breakdown →
07 / 10Persona
Persona Depth
"You're a skeptical CFO. Tear apart every weak number in this pitch."
A sharp, opinionated persona shifts the answer far more than "act as an expert."
Full breakdown →
08 / 10Templates
Prompt Templates
"Role: ___ Task: ___ Format: ___ Constraints: ___"
Build the skeleton once, fill the blanks forever — repeatable results on tap.
Full breakdown →
09 / 10Vision
Multimodal Prompting
"[attach screenshot] What's broken in this UI? List fixes, top impact first."
Prompt with images, not just text — debug, describe, or extract from any visual.
Full breakdown →
10 / 10Debug
Prompt Triage
"That missed because ___. Rewrite the prompt to fix exactly that, then redo."
A bad answer is a diagnosis — fix the prompt, not the output, and run it again.
Full breakdown →
// The One That Changes Most

Prompt triage. Everything else here makes a prompt better before you send it. Triage is the habit of treating a bad answer as information — naming why it missed, then editing the prompt rather than hitting regenerate and hoping. It's the difference between using a model and debugging one.
