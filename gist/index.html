How Context Windows Work
A language model reads text as tokens and predicts the next one from everything before it. A larger context window simply lets it hold more of that prior text in mind at once.

What it does

One highlight, four ways

Select text anywhere and a little ✦ Gist pill appears — or press the shortcut. Pick a mode; the answer streams in, TL;DR-style.

📝

Summarize

The core point plus a couple of tight bullets. The wall-of-text antidote.

💡

Explain

The same thing in plain language, jargon defined — like a smart friend put it simply.

✍️

Rewrite

A clearer, tighter version that keeps the meaning and cuts the fluff.

💬

Ask a follow-up

Type a question about the selection — "what does this mean?" — and get a grounded answer.

⌨️

⌘⇧G shortcut

Highlight and hit the shortcut to gist instantly — no pill, no clicks. Rebindable.

🎛️

Copy · Retry · Drag

Copy the result, regenerate for another take, and drag the card off whatever you're reading.

Get it running

Install in 2 minutes

Gist isn't on the Chrome Web Store — you load it directly (that's the "zero code / it's just files" part). Works in Chrome, Edge, Brave, Arc, and Opera.

Download & unzip. Grab the zip below and unzip it — you'll get a gist-extension folder. Keep it somewhere permanent.
Open chrome://extensions in your browser (paste it in the address bar).
Turn on Developer mode — the toggle in the top-right.
Click Load unpacked and select the gist-extension folder (the one with manifest.json inside).
Add your key. Click the ✦ Gist toolbar icon, paste an Anthropic API key from console.anthropic.com, pick a model (Haiku 4.5 is fast & cheap), and Save.
Try it. Highlight text on any page → click the ✦ Gist pill, or press ⌘⇧G / Ctrl+Shift+G.

// Your key, your data

Gist runs on your own Anthropic API key. It's stored locally in your browser (chrome.storage) and is sent only to Anthropic — never to us, never to the page you're on. Never paste an API key into a random website; only into the extension's popup.

When you run a gist, the text you selected is sent to Claude to generate the answer. Each gist is one API call billed to your key — Haiku 4.5 keeps it a fraction of a cent.
